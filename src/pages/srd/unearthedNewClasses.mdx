<h3><a id="generic-classes"></a>GENERIC CLASSES</h3>
<table border="0" cellpadding="0" cellspacing="0" style="width: 100%">

<td></td>

<a href="http://www.wizards.com/dnd/images/UnA_Gallery/79156.jpg"><img
class="art-wide" src="pict.jpg" title="" alt="pict"

The variety of class options available to characters can seem
overwhelming. Though that variety can lead to interesting and exciting
combinations, a game master who desires to run a simpler campaign (while
still allowing for character variety) can use these "generic" character
classes in place of the standard character classes.
<p>
The three generic classes here - <a style="color: #579eb6" href="#warrior">warrior</a>, <a style="color: #579eb6" href="#expert">expert</a>, and <a style="color: #579eb6" href="#spellcaster">spellcaster</a> - cover the
common roles of a group of adventurers. (Despite sharing names with NPC
classes, the warrior and expert presented here are very different from
those classes.) But despite these classes' basic approaches to character
building, each one allows a wide variety of archetypes through the
selection of skills and feats.
</p>
<p>
If you use these generic classes, you shouldn't also use the standard
character classes (or variants of those classes). You can still include
prestige classes, if you wish to add that level of complexity to your
game, but you may have to tweak some prestige class prerequisites that
include class features not available to these classes.
</p>
<h5><a id="saving-throws"></a>
SAVING THROWS
</h5>
Each generic class has one or two good saves and one or two poor saves. At
1st level, the character designates which saves are good or poor. If the
character later gains a level in a different class, he designates which
saves are good or poor for that class.
<p>
For example, Daniel is creating a 1st-level <a style="color: #579eb6" href="#warrior">warrior</a>, a class with one good
save and two poor saves. He wants to create a swashbuckling-type
character, so he designates his good save as Reflex and his poor saves as
Fortitude and Will.
</p>
<p>
Later, Daniel's warrior gains a level of <a style="color: #579eb6" href="#expert">expert</a>, a class with two good saves
and one poor save. He wants to keep his Reflex save high, so he designates
that as a good save. He also decides that he wants his character to become
better at resisting enchantments, so he designates Will as his other good
save, and Fortitude is left as his poor save.
</p>
<h5><a  id="class-skills"></a>
CLASS SKILLS
</h5>
Each generic class has a specific number of class skills, as given in the
class description. When a character takes his first level in a generic
class, he chooses which skills to designate as class skills. Once these
are selected, the character can't change his choice of class skills
(though if he gains a level in another class, he can choose different
skills as class skills for that class).
<p>
For example, a <a style="color: #579eb6" href="#warrior">warrior</a> has six class skills, plus <a style="color: #579eb6" href="skillsAll.html#craft">Craft</a>.
Daniel wants to play an agile, crafty warrior who uses his high Dexterity and Charisma
scores to good effect. At 1st level, the character designates his class
skills as <a style="color: #579eb6" href="skillsAll.html#bluff">Bluff</a> (Cha), <a style="color: #579eb6" href="skillsAll.html#craft">Craft</a> (Int),
<a style="color: #579eb6" href="skillsAll.html#hide">Hide</a> (Dex), <a style="color: #579eb6" href="skillsAll.html#intimidate">Intimidate</a> (Cha),
<a style="color: #579eb6" href="skillsAll.html#jump">Jump</a> (Str), <a style="color: #579eb6" href="skillsAll.html#move-silently">Move Silently</a>
(Dex), <a style="color: #579eb6" href="skillsAll.html#profession">Profession</a> (Wis), and <a style="color: #579eb6" href="skillsAll.html#tumble">Tumble</a> (Dex).
</p>
<p>
When Daniel's warrior gains a level of <a style="color: #579eb6" href="#expert">expert</a> a class with twelve class
skills plus <a style="color: #579eb6" href="skillsAll.html#craft">Craft</a> and <a style="color: #579eb6" href="skillsAll.html#profession">Profession</a>,
he must choose a new set of class skills. Daniel wants his character to be able to continue purchasing the
same skills as class skills, so he starts by designating all the skills he
chose for his warrior as class skills. Since his character has taken up
life as a thief and second story artist, he adds <a style="color: #579eb6" href="skillsAll.html#balance">Balance</a> (Dex), <a style="color: #579eb6" href="skillsAll.html#climb">Climb</a>
(Str), <a style="color: #579eb6" href="skillsAll.html#disable-device">Disable Device</a> (Int), <a style="color: #579eb6" href="skillsAll.html#knowledge">Knowledge</a> (local) (Int),
<a style="color: #579eb6" href="skillsAll.html#listen">Listen</a> (Wis), and <a style="color: #579eb6" href="skillsAll.html#open-lock">Open Lock</a> (Dex) to his list of class skills.
</p>
<h5><a id="bonus-feats"></a>
BONUS FEATS
</h5>
Each class gains a bonus feat at 1st level, and additional bonus feats at
specific levels throughout the character's career. Each time the character
gains a bonus feat, he may select any feat for which he meets the
prerequisites. There is no list of bonus feats to select from.
<p>
For the purposes of these classes, the following class features can be
selected in place of bonus feats (unless noted, each may only be selected
once).
</p>
<p><a id="evasion"></a><b>Evasion (Ex):</b> As the <a style="color: #579eb6" href="monk.html#monk-evasion">monk ability</a>. <i>Prerequisite:</i> Base Reflex save +3.</p>
<p><a id="familiar"></a><b>Familiar:</b> As the <a style="color: #579eb6" href="sorcerer.html#sorcerer-familiar">sorcerer ability</a>. Use spellcaster level to determine <a style="color: #579eb6" href="wizard.html#familiars">familiar's</a> abilities.
<i>Prerequisite:</i> ability to cast arcane spells.</p>
<p><a id="favored-enemy"></a><b>Favored Enemy (Ex):</b> As the <a style="color: #579eb6" href="ranger.html#ranger-favored-enemy">ranger ability</a>. May be selected more than once; each additional
selection improves any one favored enemy bonus (including the one just selected) by 2.</p>
<p><a id="greater-sneak-attack"></a><b>Greater Sneak Attack (Ex):</b> Add +4d6 to your sneak attack damage. <i>Prerequisites:</i> <a style="color: #579eb6" href="skillsAll.html#hide">Hide</a> 18 ranks, <a style="color: #579eb6" href="skillsAll.html#move-silently">Move
Silently</a> 18 ranks, <a style="color: #579eb6" href="#sneak-attack">sneak attack</a>, <a style="color: #579eb6" href="#improved-sneak-attack">improved sneak attack</a>.</p>
<p><a id="improved-evasion"></a><b>Improved Evasion (Ex):</b> As the <a style="color: #579eb6" href="monk.html#monk-improved-evasion">monk ability</a>. <i>Prerequisites:</i> Base Reflex save +7, <a style="color: #579eb6" href="#evasion">evasion</a>.</p>
<p><a id="improved-sneak-attack"></a><b>Improved Sneak Attack (Ex):</b> Add +3d6 to your sneak attack damage. <i>Prerequisites:</i> <a style="color: #579eb6" href="skillsAll.html#hide">Hide</a> 11 ranks, <a style="color: #579eb6" href="skillsAll.html#move-silently">Move
Silently</a> 11 ranks, <a style="color: #579eb6" href="#sneak-attack">sneak attack</a>.</p>
<p><a id="smite-evil"></a><b>Smite Evil (Su):</b> As the <a style="color: #579eb6" href="paladin.html#paladin-smite-evil">paladin ability</a>, once per day, plus one additional daily use per
five character levels. <i>Prerequisite:</i> good alignment.</p>
<p><a id="sneak-attack"></a><b>Sneak Attack (Ex):</b> As the <a style="color: #579eb6" href="rogue.html#rogue-sneak-attack">rogue ability</a>, but +2d6 on damage rolls. <i>Prerequisites:</i> <a style="color: #579eb6" href="skillsAll.html#hide">Hide</a> 4
ranks, <a style="color: #579eb6" href="skillsAll.html#move-silently">Move Silently</a> 4 ranks.</p>
<p><a id="turn-undead"></a><b>Turn undead (Su):</b> As the <a style="color: #579eb6" href="cleric.html#cleric-turn-or-rebuke-undead">cleric ability</a>. <i>Prerequisite:</i> ability to cast divine spells.</p>
<p><a id="trap-sense"></a><b>Trap Sense (Ex):</b> Combines the rogue class features <a style="color: #579eb6" href="rogue.html#rogue-trap-sense">trap sense</a> and <a style="color: #579eb6" href="rogue.html#rogue-trapfinding">trapfinding</a>.
<i>Prerequisite:</i> Search 4 ranks.</p>
<p><a id="uncanny-dodge"></a><b>Uncanny Dodge (Ex):</b> Combines the barbarian class features <a style="color: #579eb6" href="barbarian.html#barbarian-uncanny-dodge">uncanny dodge</a> and <a style="color: #579eb6" href="barbarian.html#barbarian-improved-uncanny-dodge">improved uncanny
dodge</a>. All class levels stack to determine the minimum character level required to sneak attack the character.</p>
<p><a id="wild-empathy"></a><b>Wild Empathy (Ex):</b> As the <a style="color: #579eb6" href="druid.html#druid-wild-empathy">druid ability</a>. <i>Prerequisite:</i> <a style="color: #579eb6" href="skillsAll.html#handle-animal">Handle Animal</a> 4 ranks.</p>
<p>
You can't recreate all of the standard character classes with these
generic versions, particularly classes with complicated, unique, or
specialized features such as bardic music, a wizard's familiar, or a
druid's wild shape ability. If your game master allows it, you might be
able to select other class features in place of one or more feats.
</p>

<p>
The expert can be a jack-of-all-trades or a master of a limited area of
expertise. Based on his selection of skills and feats, he can focus on
diplomatic matters, stealth, combat, wilderness survival, thievery, or any
of a number of critical tasks.
</p>
<p>
At first glance, the expert appears similar to the <a style="color: #579eb6" href="rogue.html">rogue</a>; it shares that
class's combat ability, wide range of class skills, and tendency toward
simple weapons and light armor. Unlike a rogue, however, the expert has no
special class features, but instead defines his specialty by his selection
of bonus feats.
</p>
<p><b>Hit Die:</b> d6.</p>
<p><b>Base Save Bonuses:</b> Two good saves and one poor save.</p>
<p><b>Class Skills:</b> Choose any twelve <a style="color: #579eb6" href="skillsAll.html">skills</a> as class skills, plus <a style="color: #579eb6" href="skillsAll.html#craft">Craft</a> and <a style="color: #579eb6" href="skillsAll.html#profession">Profession</a>.</p>
<p><b>Skill Points:</b> 6 + Int modifier (or four times this number at 1st level).</p>
<p><b>Weapon and Armor Proficiency:</b> All simple weapons, one martial weapon; light armor.</p>
<p><b>Bonus Feats:</b> The expert gets one bonus feat at 1st level, another at 2nd level, one at
4th level, and another one at every fourth class level thereafter (8th,
12th, and so on).</p>

<p>
Whether arcane or divine, the spellcaster has an array of magical effects
at her beck and call. Depending on her choice of spells, she can be a
healer or an enchanter, a necromancer or a trickster.
</p>
<p>
The spellcaster most closely resembles the <a style="color: #579eb6" href="sorcerer.html">sorcerer</a>, though she has fewer
spells per day than that class. In exchange, she gains a number of bonus
feats that she can use to improve her magical aptitude, increase her
survivability, heighten her combat skills, or achieve any of a wide
variety of other goals.
</p>
<p><b>Hit Die:</b> d4.</p>
<p><b>Base Save Bonuses:</b> One good save and two poor saves.</p>
<p><b>Class Skills:</b> Choose any four <a style="color: #579eb6" href="skillsAll.html">skills</a> as class skills, plus <a style="color: #579eb6" href="skillsAll.html#craft">Craft</a>, <a style="color: #579eb6" href="skillsAll.html#knowledge">Knowledge</a> (all), and
<a style="color: #579eb6" href="skillsAll.html#profession">Profession</a>.</p>
<p><b>Skill Points:</b> 2 + Int modifier (or four times this number at 1st level).</p>
<p><b>Weapon and Armor Proficiency:</b> One simple weapon; no armor.</p>
<p><b>Bonus Feats:</b> The spellcaster gets one bonus feat at 1st level, one at 5th level, and
another one at every fifth class level thereafter (10th, 15th, and so on).</p>
<p><b>Spellcasting:</b> A spellcaster learns and casts spells as a <a style="color: #579eb6" href="sorcerer.html">sorcerer</a>. She may select her
spells known from the <a style="color: #579eb6" href="clericSpells.html">cleric</a>, <a style="color: #579eb6" href="druidSpells.html">druid</a>, and <a style="color: #579eb6" href="wizardSpells.html">sorcerer/wizard</a> spell lists.</p>
<p>
A spellcaster must choose at 1st level whether to be an arcane spellcaster
or a divine spellcaster. This choice has no impact on the spells that she
may learn, but affects what kinds of scrolls she can use and which ability
score controls her spellcasting. An arcane spellcaster may designate
either Intelligence or Charisma as the ability score that determines the
highest-level spell she can learn or cast, and the Difficulty Class of her
spell's saving throws. A divine spellcaster must use Wisdom to determine
the highest-level spell she can learn or cast, and the DC of her spell's
saving throws.
</p>
<p><b>Multiclassing:</b> A character can multiclass between arcane spellcaster and divine
spellcaster.</p>

<p>
The warrior is the basic combatant of the trio of generic classes. He has
the <a style="color: #579eb6" href="fighter.html">fighter's</a> hit points, combat prowess, bonus feats, and weapon
selection.
</p>
<p>
Unlike the fighter, the warrior can choose a different array of class
skills, can designate any one of his saves as a good save, and can select
his bonus feats from the entire list of feats available. Also, if he wants
to be proficient in heavy armor, he must take the <a style="color: #579eb6" href="featsAll.html#armor-proficiency-heavy">Armor Proficiency (heavy)</a> feat.
</p>
<p><b>Hit Die:</b> d10.</p>
<p><b>Base Save Bonuses:</b> One good save and two poor saves.</p>
<p><b>Class Skills:</b> Choose any six <a style="color: #579eb6" href="skillsAll.html">skills</a> as class skills, plus <a style="color: #579eb6" href="skillsAll.html#craft">Craft</a>.</p>
<p><b>Skill Points:</b> 2 + Int modifier (or four times this number at 1st level).</p>
<p><b>Weapon and Armor Proficiency:</b> All simple and martial weapons; light and medium armor; shields (but not
tower shields).</p>
<p><b>Bonus Feats:</b>A warrior gets one bonus feat at 1st level, one at 2nd level, and another
one at every second class level (4th, 6th, and so on).</p>
<h5><a id="milticlass-generic-characters"></a>
MULTICLASS GENERIC CHARACTERS
</h5>
With only three classes to choose from (four if you count arcane
spellcasters and divine spellcasters as separate classes), multiclassing
in this variant system offers a much smaller variety of combinations than
in the standard game. Thus, the game master probably shouldn't apply an XP
penalty for characters who multiclass.
<p>
If you wish to retain the "favored class" element of the standard game,
treat any race whose favored class is barbarian, fighter, monk, or paladin
as having warrior as its favored class. A race whose favored class is
bard, ranger, or rogue has expert as its favored class. A race that has
cleric or druid as its favored class treats divine spellcaster as favored,
while a race with a favored class of sorcerer or wizard treats arcane
spellcaster as its favored class.</p>
